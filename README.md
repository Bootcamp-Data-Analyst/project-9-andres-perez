# 🧠 Clustering y Reducción de Dimensionalidad con PCA

Este proyecto presenta un análisis completo de **aprendizaje no supervisado**, aplicando técnicas de **Clustering** y **Reducción de Dimensionalidad (PCA)** para explorar la estructura interna de un conjunto de datos.

El notebook incluye el proceso completo desde la preparación de los datos hasta la visualización e interpretación de los resultados.

---

## 📌 Objetivos del Proyecto

- Comprender la estructura del dataset mediante técnicas no supervisadas.
- Aplicar algoritmos de **Clustering** para identificar grupos naturales.
- Reducir la dimensionalidad utilizando **PCA (Principal Component Analysis)**.
- Visualizar los clusters en espacios de menor dimensión.
- Interpretar los resultados obtenidos y evaluar su coherencia.

---

## 🛠 Tecnologías Utilizadas

- Python 3.x
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔎 Metodología

El flujo de trabajo del notebook sigue las siguientes etapas:

### 1️⃣ Carga y Exploración de Datos
- Inspección del dataset
- Análisis descriptivo
- Identificación de posibles patrones

### 2️⃣ Preprocesamiento
- Limpieza de datos
- Escalado de variables
- Preparación para modelado

### 3️⃣ Reducción de Dimensionalidad
- Aplicación de **PCA**
- Análisis de varianza explicada
- Selección de componentes principales

### 4️⃣ Clustering
- Implementación de algoritmos como:
  - K-Means
  - (Otros si aplican en el notebook)
- Determinación del número óptimo de clusters (método del codo / silhouette)

### 5️⃣ Visualización
- Representación gráfica de clusters
- Visualización en 2D usando componentes principales

---

## 📊 Resultados

- Identificación de agrupaciones naturales en los datos.
- Reducción significativa de la dimensionalidad manteniendo la mayor varianza posible.
- Visualizaciones claras que permiten interpretar la separación entre grupos.
